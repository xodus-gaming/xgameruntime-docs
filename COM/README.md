# COM Classes

The classes below are provided by [`QueryApiImpl`](../QueryApiImpl.md).
The functions provided by `xgameruntime.lib` mostly match one to one with an
entry in one of their respective classes' interfaces. Additionally, many
functions provided by `xgameruntime.lib` will also call
[XThreadAssertNotTimeSensitive](https://learn.microsoft.com/xbox/gdk/docs/reference/system/xthread/functions/xthreadassertnottimesensitive).
Our information about these classes and their interfaces is incomplete, some of
them contain unknown methods and have missing interface ids, however all
interface ids used since the first public tag of GDK are documented here.
